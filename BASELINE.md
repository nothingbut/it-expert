## 配置参数

### WSL2 配置（计划值）

#### .wslconfig 文件
位置: `%USERPROFILE%\.wslconfig`

```ini
[wsl2]
memory=20GB                    # 内存限制（总内存64G，分配20G）
processors=8                   # CPU核心数（AMD 7735为8核16线程）
autoMemoryReclaim=gradual      # 自动内存回收
swap=8GB                       # 交换文件大小
swapFile=D:\\WSL\\swap.vhdx    # 交换文件路径（D盘）
networkingMode=NAT             # 网络模式
```

#### Ubuntu 安装参数
- **发行版**: Ubuntu 24.04 LTS
- **安装路径**: D:\WSL\Ubuntu
- **WSL 版本**: 2
