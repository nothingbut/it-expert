## 🎯 2 步快速部署

### 第 1 步：服务端配置（铭凡 UM773）⏱️ 2 分钟

#### 1.1 下载安装 RustDesk

```
1. 打开浏览器
2. 访问：https://rustdesk.com/
   或：https://github.com/rustdesk/rustdesk/releases
3. 下载最新版：rustdesk-x.x.x-x86_64.exe（约 20MB）
4. 运行安装程序
5. 安装完成后自动启动
```

#### 1.2 记录连接信息

```
RustDesk 窗口中央显示：

您的桌面 ID: 123456789 ← 记录这个 ID

密码: [点击设置密码]
```

**设置密码**（重要！）：
```
1. 点击密码右侧的 "…" 按钮
2. 设置密码（例如：MySecure123）
3. ✅ 勾选 "使用永久密码"
4. 确定
```

#### 1.3 配置无人值守访问（推荐）

```
1. 点击右上角 "设置" 图标 ⚙️
2. 安全 → 权限：
   - ✅ 启用 "允许键盘/鼠标"
   - ✅ 启用 "允许剪贴板"
   - ✅ 启用 "允许文件传输"
   - ✅ 启用 "允许音频"
3. 高级：
   - ✅ 启用 "随系统启动"
   - ✅ 启用 "自动接受连接"（可选，方便但降低安全性）
4. 关闭设置
```

✅ 服务端配置完成！

---

### 第 2 步：客户端配置（联想 ThinkBook+）⏱️ 2 分钟

#### 2.1 下载安装 RustDesk

```
1. 访问：https://rustdesk.com/
2. 下载 Windows 版本
3. 安装并启动 RustDesk
```

#### 2.2 连接铭凡

**方式 1: 快速连接**
```
1. RustDesk 窗口中输入铭凡的 ID：123456789
2. 点击 "连接"
3. 输入密码：MySecure123
4. 连接成功 ✅
```

**方式 2: 地址簿（推荐，更方便）**
```
1. 点击左侧 "地址簿" 📖
2. 点击 "+" 添加新连接
3. 填写信息：
   - ID: 123456789
   - 别名: 铭凡 UM773
   - 用户名: （留空）
   - 密码: MySecure123
   - ✅ 勾选 "保存密码"
4. 保存
5. 以后双击 "铭凡 UM773" 即可连接
```

✅ 完成！现在可以远程访问了

## 🔧 常见问题

### ❓ 无法连接

**原因**：防火墙阻止或 ID 错误

**解决**：
```
1. 检查 ID 是否正确
2. 检查密码是否正确
3. 铭凡上 RustDesk 是否运行（托盘查看）
4. Windows 防火墙 → 允许 RustDesk
```

---

### ❓ 延迟很高

**原因**：使用了公共中继服务器

**解决**：
```
1. 连接后右键 → 显示
2. ✅ 确认 "直接连接" 已启用
3. 查看状态栏：应显示 "Direct"
4. 如显示 "Relay"，检查网络连通性
```

**检查方法**：
```powershell
# ThinkBook+ 上测试连通性
ping 192.168.x.x  # 铭凡 IP
```

---

### ❓ 画面模糊/卡顿

**解决**：
```
1. 降低分辨率：1080p → 720p
2. 降低帧率：60 → 30 fps
3. 关闭铭凡上其他占用 CPU 的程序
4. 确保千兆局域网连接
```

---

### ❓ 鼠标/键盘不响应

**原因**：权限未启用

**解决**（铭凡上）：
```
1. RustDesk → 设置
2. 安全 → 权限
3. ✅ 确认 "允许键盘/鼠标" 已勾选
4. 重新连接
```
